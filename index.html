GitHub Pages (HTML) থেকে API রেসপন্স দেখা

নোট: GitHub Pages এ PHP রান হয় না। তাই এই পেজটা JavaScript দিয়ে সরাসরি upstream URL এ request করে রেসপন্স দেখায়। যদি upstream এ CORS বন্ধ থাকে তাহলে ব্রাউজার ব্লক করবে—সেক্ষেত্রে proxy (Cloudflare Worker/Vercel) লাগবে।
